Overview

During military service, I noticed a repeated manual calculation workflow that required careful coordinate input, visual reading, and repeated verification. I converted the repeatable parts of the workflow into a simple Excel-based tool to improve consistency and reduce manual effort during training documentation.

My Contribution

  • Mapped the manual workflow into input, calculation, and output steps.
  • Built a small Excel interface for coordinate-style inputs and computed outputs.
  • Implemented VBA logic behind a single button-driven workflow.
  • Focused on repeatability, error reduction, and clear output formatting.

Implementation

Sanitized VBA structure

The public portfolio version keeps the VBA structure and automation approach visible, but intentionally removes operational formulas and sensitive calculation details.

Sub CalculateFDC()

    Dim X As Double
    Dim Y As Double
    Dim OutputA As Double
    Dim OutputB As Double

    X = Range("B4").Value
    Y = Range("B5").Value

    ' Calculation model intentionally omitted
    ' in the public portfolio version.

    OutputA = CalculateOutputA(X, Y)
    OutputB = CalculateOutputB(X, Y)

    Range("B8").Value = OutputA
    Range("B9").Value = OutputB

End Sub
